## :memo: How to use ?
1. First clone the project.

```bash
git clone https://github.com/adrielborges/list-users
```
2. Enter in file of project.

```bash
  cd list-users
```
3. install dependencies.

* **npm**
```bash
npm install
```
* **yarn**
```bash
yarn
```

4. To emulate the app:
* **npm**
```bash
npm start
```
* **yarn**
```bash
yarn start
```
